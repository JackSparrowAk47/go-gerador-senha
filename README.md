# 🔐 Go Gerador de Senhas

Gerador de senhas aleatórias desenvolvido em Go (Golang), permitindo criar senhas seguras e personalizadas com diferentes critérios.

## 🚀 Funcionalidades

- Gerar senhas aleatórias
- Definir tamanho da senha
- Incluir letras maiúsculas
- Incluir letras minúsculas
- Incluir números
- Incluir caracteres especiais
- Interface simples via terminal

---

## 🛠️ Tecnologias utilizadas

- Go (Golang)

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/JackSparrowAk47/go-gerador-senha.git
```

Entre na pasta do projeto:

```bash
cd go-gerador-senha
```

Execute o projeto:

```bash
go run main.go
```

---

## 💻 Exemplo de uso

```bash
Digite o tamanho da senha: 16

Senha gerada:
A@9x#Lm2!Pq8$Zr
```

## 🔒 Sobre segurança

As senhas são geradas de forma aleatória utilizando recursos da linguagem Go para aumentar a segurança e imprevisibilidade.
